# Code provenance

This focused repository was extracted on 2026-07-20 from commit `f678565` of
the authors' larger BioFly development workspace. The scientific functions were
copied without changing their model equations. Release-specific changes were
limited to portable repository-relative paths, packaging, dependency metadata,
source-data validation, figure entry points and documentation.

The full-graph simulations recorded the following upstream revisions in their
run metadata:

- FlyWire annotations: `f02717b7624e502f4c72443e986976feff2b563e`
- Shiu whole-brain model: `91bdd1e7dcf193f3e7ca5a8933497fcef63b7960`

The compact source-data tables are byte-verified by
`data/source_data.sha256`. Model reruns should record input file hashes in their
output metadata because upstream archives may publish revised files under new
versions.
